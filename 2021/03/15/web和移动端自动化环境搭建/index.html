appium、selenium环境搭建

一、环境配置

1.1 python安装配置

python官网下载并安装Python3，（下载网址https://www.python.org/downloads/）
python环境变量配置
D:\program\python37
D:\program\python37\Scrippts

检查是否安装成功：
命令行输入：python,查看是否有版本输出

1.2pip安装

PIP: 方便安装python依赖包， 直接通过：pip install 包名， 从相关镜像下载
检查是否已默认安装：
命令行运行：pip install selenium, 运行正常

注意：后续python包都通过命令行：pip install 包名

若未安装，windows安装方法:
一、首先安装setuptool：
https://pypi.org/project/setuptools/#files  从该网址下载zip，并解压；打开cmd，cd到解压目录，运行命令：python setup.py install
二、接下来安装pip:
https://pypi.org/project/pip/#files 该网址下载tar文件，并解压打开cmd，cd到解压目录，运行命令:python setup.py install

1.3 python开发工具安装pycharm

下载地址：https://www.jetbrains.com/pycharm/download/#section=windows，通常我们使用community版本即可

1.4 下载并安装JDK

如 Java SE Development Kit 8u211
根据安装路径配置java环境变量
JAVE_HOME
%JAVE_HOME%\bin
%JAVE_HOME%\jre\bin

1.5 web自动化必备

1.5.1 chrome浏览器和chromedriver下载

1、下载chrome浏览器
2、根据chrome浏览器的版本下载相应的chromedriver, 参考映射关系：https://blog.csdn.net/BinGISer/article/details/88559532

3、将chromedriver放到项目相应的路径或者python目录下

1.5.2 安装web自动化python相关包

主要包名有：selenium、ruamel.yaml等， 也可以通过运行报错安装需要的包。
安装方式：进入命令行，pip install 包名

1.6 移动端自动化必备

1.6.1 下载并安装Android SDK

1、下载并安装android-studio
2、安装相关SDK（也可以直接下载sdk安装，参考https://blog.csdn.net/qq_27011361/article/details/79490588）
安装相应的包
根据手机版本安装SDK platform

配置环境变量
ANDROID_ADB_SEVER_PORT 5037
ANDORID_HOME …\android-sdk
%ANDORID_HOME%\tools
%ANDORID_HOME%\platform-tools

1.6.2appium安装

Appium是一个开源、跨平台的测试框架，可以用来测试原生及混合的移动端应用。可以跨平台同时支持Android、iOS。支持多种语言，java、python、php、Ruby等等。
安装移动端开源试框架appium，需要安装server端和client端
1、appium server端， 安装appium-desktop，可选择最新版本一键安装。
下载地址：http://appium.io/
2、appium client端，通过命令行安装包：
pip install Appium-Python-Client

二、web端元素定位方法

1、打开chrome浏览器
2、右键点击检查
3、点击小箭头
4、再选择需要定位的元素，可获取到元素信息
5、定位方式可通过class, id , xpath, css等

三、移动端元素定位方法

3.1 检查手机是否连接
打开手机USB调试，连上手机，通过adb devices查看手机是否连上

3.2 元素定位方法
方法1：可点击..\Android\android-sdk\tools\ monitor.bat
点击如下图第一个标记框，会将手机页面显示，点击对应的元素，最右边出现元素信息
（注意：小程序端属性基本一致，可通过文本定位、数组、包含某些字符，最坏通过坐标定位）
